Модульный набор плагинов и компонентов
для удобной разработки интерфейсов на Vue и React

Устали от переопределения стилей базовых компонентов
и топорности высокоуровневых UI библиотек?

npm install blanc-ui@latest

Самобытный

Все модули реализованы самостоятельно. В клиентские зависимости допущены только низкоуровневые утилиты, а значит если нужно кастомизировать логику - сделайте форк и вы сможете кастомизировать всё.

Абстрактный

Из коробки содержит минимум стилей, необходимых для отображения компонента. Все стили наследуются компонентами через переопределяемые свойства. Их отображение можно подстраивать под любой дизайн.

Масштабируемый

Содержит только низкоуровневые компоненты с базовой логикой. Рендер под Vue и React написан на jsx и предельно идентичен. Насколько это было, конечно, возможно.

Полностью типизированный

Все модули проекта покрыты типами. В типизации не использованы жёсткие связи и наследования. Все типы модулей самобытны, это позволяет избежать лишних проблем при их расширении.
